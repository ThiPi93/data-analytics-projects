# Portfólio de Projetos em Análise de Dados

Este repositório reúne projetos de análise de dados desenvolvidos com foco em
portfólio profissional. Os trabalhos aqui apresentados exploram dados reais,
aplicam métodos estatísticos, visualização de dados e análises exploratórias,
sempre com preocupação em clareza, reprodutibilidade e interpretação correta
dos resultados.

Os projetos não têm caráter preditivo ou comercial, sendo voltados para fins
educacionais, demonstração técnica e divulgação profissional.

**Principais tópicos abordados:**
- Análise exploratória de dados (EDA)
- Estatística descritiva e inferencial
- Testes estatísticos
- Visualização de dados
- Python aplicado à análise de dados

# Data Analysis Portfolio Projects

This repository contains data analysis projects developed for professional
portfolio purposes. The projects explore real-world datasets and apply
statistical methods, data visualization, and exploratory data analysis, with
a strong focus on clarity, reproducibility, and proper interpretation of
results.

The projects are non-predictive and non-commercial, intended for educational,
technical demonstration, and professional showcase purposes.

**Main topics covered:**
- Exploratory Data Analysis (EDA)
- Descriptive and inferential statistics
- Statistical hypothesis testing
- Data visualization
- Python for data analysis
